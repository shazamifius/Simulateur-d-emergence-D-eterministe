# Politique de sécurité

## Signalement d'une vulnérabilité

Nous prenons la sécurité de notre projet au sérieux. Si vous découvrez une vulnérabilité de sécurité dans ce projet, veuillez ne pas signaler le problème publiquement via le gestionnaire de tickets GitHub.

Au lieu de cela, veuillez contacter le responsable en envoyant un e-mail à l'adresse fournie sur le profil GitHub de l'auteur (`shazamifius@gmail.com`).

Nous vous répondrons dans les plus brefs délais pour accuser réception de votre rapport et coordonner la correction de la vulnérabilité de manière responsable.
